Shows a random Chuck Norris joke on your blog, possibly starring yourself.

== Description ==

This plugin adds a small Wordpress widget that shows a random Chuck Norris joke on your blog.
These jokes are taken from a database on chuck.maartendecat.be. It is also possible to change
the name of the main character in the joke to star in it yourself!

== Installation ==

To install this plugin:

1. Upload `chuck-norris-jokes.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag the "Chuck Norris Joke" widget in a widget area through the 'Appearance - Widgets' menu.
1. Adjust the name of the main character in the joke or keep the default settings. Then press save.

You should now see a different Chuck Norris joke on your blog every time the page loads.
